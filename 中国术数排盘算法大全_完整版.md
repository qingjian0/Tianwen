# 中国术数排盘算法大全

## 一、梅花易数 (Meihua Yi Shu)

### 1.1 起卦方法

#### 1.1.1 时间起卦法
**核心公式：**
- 上卦 = (年支序数 + 农历月 + 农历日) ÷ 8 取余数
- 下卦 = (年支序数 + 农历月 + 农历日 + 时辰序数) ÷ 8 取余数
- 动爻 = (年支序数 + 农历月 + 农历日 + 时辰序数) ÷ 6 取余数

**年支序数：**
子(1)、丑(2)、寅(3)、卯(4)、辰(5)、巳(6)、午(7)、未(8)、申(9)、酉(10)、戌(11)、亥(12)

**时辰序数：**
子(1)、丑(2)、寅(3)、卯(4)、辰(5)、巳(6)、午(7)、未(8)、申(9)、酉(10)、戌(11)、亥(12)

**先天八卦数：**
乾(1)、兑(2)、离(3)、震(4)、巽(5)、坎(6)、艮(7)、坤(8)

#### 1.1.2 数字起卦法
- **单数字：** 上卦 = 数字 ÷ 8，下卦 = (数字+1) ÷ 8，动爻 = 数字 ÷ 6
- **双数字：** 上卦 = 第一个数 ÷ 8，下卦 = 第二个数 ÷ 8，动爻 = (两数之和) ÷ 6
- **三数字：** 上卦 = 第一个数 ÷ 8，下卦 = 第二个数 ÷ 8，动爻 = 第三个数 ÷ 6

#### 1.1.3 铜钱起卦法
- 三枚铜钱抛六次，记录正反面
- 阳爻(O)：2正1反或3正
- 阴爻(X)：2反1正或3反
- 动爻：变爻标记

### 1.2 卦象计算

#### 1.2.1 互卦
- 取本卦第2、3、4爻为下互卦
- 取本卦第3、4、5爻为上互卦

#### 1.2.2 变卦
- 根据动爻位置变换阴阳属性
- 阳爻变阴爻，阴爻变阳爻

#### 1.2.3 错卦与综卦
- **错卦：** 所有爻取反
- **综卦：** 卦象倒置

### 1.3 体用关系
| 情况 | 体卦 | 用卦 |
|------|------|------|
| 无动爻 | 下卦 | 上卦 |
| 下卦动 | 上卦 | 下卦 |
| 上卦动 | 下卦 | 上卦 |

### 1.4 五行生克
- 生：金生水、水生木、木生火、火生土、土生金
- 克：金克木、木克土、土克水、水克火、火克金

## 二、奇门遁甲 (Qi Men Dun Jia)

### 2.1 起局方法

#### 2.1.1 确定局数
**拆补法：**
```
根据节气确定阴阳遁：
- 冬至后：阳遁（一至九局）
- 夏至后：阴遁（九至一局）

节气与局数对应：
- 冬至、小寒：阳遁1-2局
- 大寒、立春：阳遁3-4局
- 雨水、惊蛰：阳遁5-6局
- 春分、清明：阳遁7-8局
- 谷雨、立夏：阳遁9局
- 夏至、小暑：阴遁9-8局
- 大暑、立秋：阴遁7-6局
- 处暑、白露：阴遁5-4局
- 秋分、寒露：阴遁3-2局
- 霜降、立冬：阴遁1局
```

## 三、大六壬 (Da Liu Ren)

### 3.1 起课方法

#### 3.1.1 确定月将
```
月将 = 节气对应的天将
正月雨水后：亥将（登明）
二月春分后：戌将（河魁）
三月谷雨后：酉将（从魁）
四月小满后：申将（传送）
五月夏至后：未将（小吉）
六月大暑后：午将（胜光）
七月处暑后：巳将（太乙）
八月秋分后：辰将（天罡）
九月霜降后：卯将（太冲）
十月小雪后：寅将（功曹）
十一月冬至后：丑将（大吉）
十二月大寒后：子将（神后）
```

#### 3.1.2 定地盘与天盘
- **地盘：** 固定十二地支方位
- **天盘：** 月将加时支，顺布十二天将

#### 3.1.3 四课
```
第一课（干上）：日干 + 天盘上的天将
第二课（支上）：日支 + 天盘上的天将
第三课（干阴）：日干遁干 + 天盘上的天将
第四课（支阴）：日支遁干 + 天盘上的天将
```

#### 3.1.4 三传
**起传规则：**
1. **贼克法：** 取相克者为初传
2. **比用法：** 无相克则取相比者
3. **涉害法：** 取受克深者
4. **遥克法：** 取隔位相克者
5. **昴星法：** 特殊规则

```typescript
// 年干支
function getYearGanZhi(year: number): string {
  const gan = ["甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸"];
  const zhi = ["子", "丑", "寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥"];
  return gan[(year - 4) % 10] + zhi[(year - 4) % 12];
}

// 月干支（五虎遁）
function getMonthGanZhi(yearGan: string, month: number): string {
  const ganOrder = ["甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸"];
  const zhiOrder = ["寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥", "子", "丑"];
  
  const baseIdx = {
    "甲": 2, "己": 2,
    "乙": 4, "庚": 4,
    "丙": 6, "辛": 6,
    "丁": 8, "壬": 8,
    "戊": 0, "癸": 0,
  }[yearGan] || 0;
  
  const ganIdx = (baseIdx + month - 1) % 10;
  return ganOrder[ganIdx] + zhiOrder[month - 1];
}
```
