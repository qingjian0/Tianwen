天问系统 API 文档

RESTful + WebSocket API

版本: v1.0 | Base URL: http://localhost:3000/api | WebSocket: ws://localhost:3000/ws

1. 概览

天问系统提供 RESTful API 进行预测、规则查询、系统管理，同时提供 WebSocket 实时推送接口，用于获取推演进度、规则更新和系统状态。

Base URL
http://localhost:3000/api
WebSocket URL
ws://localhost:3000/ws

2. RESTful API

2.1 预测接口 (Prediction)

POST /predictions 创建新的预测请求
请求体:
{
  "question": "今日财运如何？",
  "category": "wealth",
  "system": "meihua",
  "mode": "single"
}
响应 (202 Accepted):
{
  "id": "prediction_12345",
  "status": "processing",
  "submittedAt": "2026-05-19T10:00:00Z",
  "estimatedCompletion": "2026-05-19T10:00:05Z"
}
category 可选值: wealth, career, love, market | system 可选值: meihua, liuyao, bazi, qimen, ziwei

2.2 规则接口 (Rule)

GET /rules
获取所有规则列表
GET /rules/:id
获取指定规则详细信息
GET /rules/categories/list
获取规则分类列表
GET /rules/categories/:category
获取某分类规则

2.3 系统接口

GET /health
健康检查
GET /version
系统版本信息

3. WebSocket 实时推送

3.1 连接 URL

ws://localhost:3000/ws

3.2 频道 (Channels)

predictions
推演状态更新与完成通知
rules
规则库更新通知
system
系统状态与健康监控

4. 错误码约定

HTTP 状态码

200 成功
202 请求已接受，处理中
400 请求参数错误
401 未授权
404 资源不存在
500 服务器内部错误

WebSocket 错误消息

{
  "channel": "system",
  "event": "error",
  "data": { "code": 5001, "message": "Pipeline 执行失败" },
  "timestamp": "2026-05-19T10:00:02Z"
}

5. 使用示例

5.1 使用 RESTful API

import axios from 'axios';

const response = await axios.post('http://localhost:3000/api/predictions', {
  question: '今日财运如何？',
  category: 'wealth',
  system: 'meihua',
  mode: 'single'
});

console.log('预测请求ID:', response.data.id);

5.2 使用 WebSocket

import WebSocket from 'ws';

const ws = new WebSocket('ws://localhost:3000/ws');

ws.on('open', () => {
  console.log('WebSocket 已连接');
  
  // 订阅预测频道
  ws.send(JSON.stringify({
    action: 'subscribe',
    channel: 'predictions'
  }));
});

ws.on('message', (message) => {
  const msg = JSON.parse(message.toString());
  console.log('收到消息:', msg);
});
